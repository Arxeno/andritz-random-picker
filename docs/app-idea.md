This is a doorprize management system for family gathering event. where it has these features:

1. Login page, where it's the page to login to the system. The user is hardcoded in the system. Only 1 user is allowed to login. Only Admin.
2. Main menu page, where it's the main page and has navigations to other pages.
3. Spin the wheel page, where it's the page to spin the wheel and get the prize.
4. Saving the winner of the doorprize into the database.
5. Admin can export the winner list into a xlsx file.
6. Admin can add participants into the system. Can add manually or import from a xlsx file. Only Full Name, email, and phone number is required.
7. Admin can do re-spin multiple times before confirming the winner.
8. Admin can view the winner history.
